# MiniMind 角色功能使用指南

## 概述

MiniMind Text Generator 节点现在支持多种角色定位，让AI能够根据不同的专业角色提供更精准的回答。

## 可用角色

### 1. 通用助手
- **描述**: 默认角色，能够回答各种问题并提供帮助
- **适用场景**: 日常对话、一般性问题咨询
- **示例输入**: "你好，请介绍一下你自己。"
- **预期输出**: 友好的自我介绍和能力说明

### 2. 翻译专家
- **描述**: 专业翻译，能够准确地在中英文之间进行翻译
- **适用场景**: 文本翻译、语言转换
- **示例输入**: 
  - "Hello, how are you today?" (英译中)
  - "你好，今天天气怎么样？" (中译英)
- **预期输出**: 直接提供翻译结果，无需额外解释

### 3. 提示词扩写师
- **描述**: 将简短的提示词扩展成详细、具体、富有创意的完整提示词
- **适用场景**: AI绘画提示词优化、创意写作提示
- **示例输入**: "一只猫在花园里"
- **预期输出**: "一只优雅的橘色长毛猫，正悠闲地漫步在春日的花园中，周围盛开着五彩斑斓的花朵，阳光透过绿叶洒下斑驳的光影，营造出温馨宁静的午后时光..."

### 4. 创意写作师
- **描述**: 富有创意的写作师，擅长创作各种类型的文学作品
- **适用场景**: 小说创作、诗歌写作、剧本编写
- **示例输入**: "写一首关于春天的短诗"
- **预期输出**: 生动有趣、富有想象力的诗歌作品

### 5. 技术专家
- **描述**: 精通编程、软件开发、系统架构等技术领域
- **适用场景**: 技术问题解答、代码优化建议、架构设计
- **示例输入**: "解释一下什么是机器学习"
- **预期输出**: 准确的技术解释和清晰易懂的概念说明

### 6. 教学助手
- **描述**: 擅长用简单易懂的方式解释复杂概念
- **适用场景**: 学习指导、知识解释、教学辅助
- **示例输入**: "如何学习编程？"
- **预期输出**: 结构化的学习建议和循序渐进的指导方案

## 使用方法

### 在ComfyUI中使用

1. 添加 "MiniMind Text Generator" 节点到工作流
2. 在 "role" 参数中选择合适的角色
3. 在 "prompt" 中输入你的问题或需求
4. 调整其他参数（如 max_length、temperature 等）
5. 运行工作流获取结果

### 参数说明

- **prompt**: 输入文本或问题
- **role**: 选择AI角色（通用助手、翻译专家、提示词扩写师、创意写作师、技术专家、教学助手）
- **max_length**: 生成文本的最大长度（1-2048）
- **temperature**: 创造性控制（0.1-2.0，越高越有创意）
- **top_p**: 词汇选择范围（0.1-1.0）
- **do_sample**: 是否使用采样生成
- **repetition_penalty**: 重复惩罚（1.0-2.0，避免重复内容）

## 最佳实践

### 1. 角色选择建议
- 根据任务类型选择最合适的角色
- 翻译任务使用"翻译专家"
- 创意内容使用"创意写作师"或"提示词扩写师"
- 技术问题使用"技术专家"
- 学习相关使用"教学助手"

### 2. 参数调优
- **翻译任务**: 较低的temperature（0.3-0.5）确保准确性
- **创意写作**: 较高的temperature（0.7-1.0）增加创造性
- **技术解答**: 中等temperature（0.5-0.7）平衡准确性和表达

### 3. 提示词技巧
- 明确具体的需求
- 提供必要的上下文信息
- 对于翻译任务，直接提供需要翻译的文本
- 对于扩写任务，提供核心关键词或简短描述

## 示例工作流

### 翻译工作流
```
输入: "Good morning, have a nice day!"
角色: 翻译专家
参数: temperature=0.3, max_length=100
输出: "早上好，祝你今天愉快！"
```

### 提示词扩写工作流
```
输入: "科幻城市"
角色: 提示词扩写师
参数: temperature=0.8, max_length=200
输出: "未来科幻城市，高耸入云的摩天大楼，霓虹灯闪烁，飞行汽车穿梭其间，全息广告投影在空中，机器人和人类和谐共存，充满科技感的建筑风格，蓝紫色调的夜景氛围..."
```

### 创意写作工作流
```
输入: "写一个关于时间旅行的短故事开头"
角色: 创意写作师
参数: temperature=0.9, max_length=300
输出: 富有想象力的故事开头
```

## 注意事项

1. **模型加载**: 首次使用时需要加载模型，可能需要几分钟时间
2. **内存管理**: 大型模型会占用较多内存，建议在不使用时卸载模型
3. **生成质量**: 不同角色的输出质量可能因模型训练数据而有所差异
4. **语言支持**: 主要支持中文和英文，其他语言支持有限

## 故障排除

- **模型加载失败**: 检查模型路径是否正确
- **生成结果不理想**: 尝试调整temperature和角色选择
- **内存不足**: 降低max_length或重启应用释放内存
- **角色效果不明显**: 确保提示词与角色定位匹配
